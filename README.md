# Счёт (Schet)

Android-приложение: математические примеры, вход через Google, облачная статистика игроков, автопроверка обновлений с GitHub Releases.

## Возможности

- Примеры `a + b` и `a − b` (числа до 100)
- Вход через **Google**
- Общий список игроков и статистика в **Firebase** (без компьютера)
- При запуске проверка новой версии APK в этом репозитории

## Быстрый старт

1. Настройте Firebase по инструкции [`FIREBASE_SETUP.md`](./FIREBASE_SETUP.md)
2. Заполните `src/config.ts`
3. Положите `google-services.json` в корень проекта
4. Соберите APK и загрузите в Releases как `Schet.apk`

## Релизы

Тег формата `v1.1.0`. В assets обязательно файл **`Schet.apk`**.
